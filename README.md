# AI Control Tower - Open-Source Portfolio Control Plane

Cross-project operating repo for Henry's four-agent OpenClaw system.

## Purpose

- hold the portfolio-level roadmap
- define how Athena, Hermes, Hephaestus, and Themis collaborate
- track the three MVP tracks: `Agora`, `Physis`, and `Titan`
- keep decisions and operating rules in one durable place

## Current Priority Order

1. `Agora`
2. `Physis`
3. `Titan`

## Repository Layout

- `docs/portfolio-overview.md`
- `docs/operating-model.md`
- `docs/projects/agora.md`
- `docs/projects/physis.md`
- `docs/projects/titan.md`
- `docs/workflows/agent-loop.md`

## Initial Goal

Establish a clean control plane for the multi-agent system before heavy implementation begins.
