# Third-Party Code Policy

## Why This Exists

This portfolio is open-source-first, but open-source intent does not automatically grant permission to copy outside code, assets, or datasets.

## Rules

1. Do not copy code from repositories or websites that have no clear license.
2. Prefer permissive sources first: Apache-2.0, MIT, BSD, ISC.
3. If code is reused, record the original source URL, license, and what was changed.
4. Preserve required attribution, NOTICE text, and copyright headers.
5. Treat GPL/AGPL/LGPL code as a separate legal review topic before incorporation.
6. Prefer dependencies, references, adapters, or clean-room rewrites over bulk copying.
7. For datasets, models, images, and docs, check their terms separately; software license rules do not automatically cover them.

## Practical Operating Rule

When in doubt:

- link to the source
- summarize the idea
- reimplement only what is clearly safe to reimplement

Do not assume "non-commercial for now" removes infringement risk. It does not.
