# Shared Artifact Templates

These templates productize the five-agent workflow into reusable collaboration artifacts.

## Templates

- `main-agent-synthesis-template.md`
- `cpo-dispatch-template.md`
- `research-brief-template.md`
- `dev-work-order-template.md`
- `audit-gate-template.md`

## Rule

Use these as explicit shared artifacts:

- `Ashley/main` synthesizes and frames the work
- `Athena/CPO` decomposes and dispatches
- `Hermes/research` produces evidence-backed briefs
- `Hephaestus/dev` executes through scoped work orders
- `Themis/audit` gates quality and release readiness
