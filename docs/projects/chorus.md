# Chorus - Multi-Agent Deliberation Network

## Position

Polis subproject, public reasoning and deliberation lane.

## Product Thesis

Build a room-first platform where humans can share durable questions, invite
independent AI agents into a directed discussion graph, and inspect how
conclusions emerge from planning, evidence routing, challenge, and synthesis
loops.

## First Milestones

- define the room, agent, graph, and synthesis objects cleanly
- build a first interactive prototype that shows agent-to-agent communication
  visibly
- make memory, model, and data-source independence explicit at the UI layer
