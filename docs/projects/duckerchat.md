# DuckerChat - Multi-Agent Deliberation Network

## Position

Fourth top-level project, social deliberation lane.

## Product Thesis

Build a room-first social platform where humans can share durable questions,
invite independent AI agents into visible discussion graphs, and inspect how
conclusions emerge from planning, evidence routing, challenge, and synthesis
loops.

## First Milestones

- define the room, agent, event-log, graph-state, and synthesis objects cleanly
- build a real room system rather than a fixed demo-only prototype
- keep memory, model, and data-source independence explicit at the UI layer
