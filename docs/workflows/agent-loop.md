# Agent Loop

Henry -> Ashley/main -> Athena -> Hermes -> Hephaestus -> Themis -> Henry

- Athena writes plans and milestones.
- Hermes writes research briefs with sources and dates.
- Hephaestus ships code, scripts, and repo structure.

- failed audit blocks release
- scope drift returns to Athena
